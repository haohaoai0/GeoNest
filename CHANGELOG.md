# GeoNest 版本更新日志

## v1.0.3（2026-07-17）

> 本日志基于当前本地工作区与 GitHub 首个公开版本 `v1.0.0`（`e769f71`，2026-07-05）的实际代码差异整理。当前变更尚未提交至 GitHub。

### 版本亮点

- Processing 算法目录由 22 项扩展至 128 项，覆盖几何处理、叠加分析、邻近分析、空间统计、栅格分析、制图综合、时空分析和数据管理等场景。
- 新增后台处理任务体系，支持进度与阶段显示、任务取消、系统通知、处理历史记录及结果自动加载。
- 新增 QGIS 工程读写能力，可打开和保存 `.qgs`、`.qgz` 工程，并支持 ZIP/KMZ 等归档数据解压导入。
- 新增拓扑规则校验与修复工作流，支持全量或增量检查、问题定位、修复预览、提交、回滚和复检。
- 新增华为账号登录、云文件同步和桌面服务卡片，便于查看当前工程、处理状态与地图缩略图。

### 新增功能

#### GIS 分析与数据处理

- 新增统一的 Processing 工具浏览器，支持算法分类、搜索、运行入口、活动任务和历史记录查看。
- 新增高级空间分析能力，包括空间中心、空间自相关、热点分析、空间权重、空间回归、空间预测、时空聚类、时间序列预测与异常检测等。
- 新增常用几何与叠加工具，包括凸包、最小外包矩形、拆分多部件、要素转点/线/面、节点转点、加密、平滑、去孔洞、联合、标识、更新、对称差异和多环缓冲区等。
- 新增捕捉、整合、延伸/修剪线、中心线、渔网、六边形网格、随机点、规则采样点、高级缓冲区、高级裁剪和空间连接等工具。
- 新增核密度、点密度、线密度、IDW 插值、坡度、坡向和栅格重分类等栅格处理能力。
- 新增摘要统计、频数、唯一值、分组统计、交叉表、统计图表和多边形邻接分析。
- 新增后台任务准备与独立执行机制，避免长耗时计算阻塞主界面，并支持取消、进度回调和任务资源释放。
- 新增处理输出目录、覆盖策略、结果自动加载、历史保留开关及历史数量上限设置。

#### 数据导入、转换与交付

- 扩展文件打开和系统分享导入范围，支持 Shapefile 关联文件、GeoJSON、GeoPackage、KML/KMZ、CSV、XLS/XLSX、TIFF、JPEG、PNG 等格式。
- 新增拖放导入，可将外部 GIS 文件直接拖入工作台，也可通过拖放调整图层顺序。
- 新增 XY 表转点工具，支持 CSV 分隔符配置、XLS/XLSX 工作表选择、X/Y/Z/M 字段识别、数据预览和坐标系设置。
- 原生 GDAL 构建加入 FreeXL，补充旧版 `.xls` 表格读取支持。
- 新增多格式图层导出，支持 Shapefile、GeoJSON、GeoPackage、KML、CSV、FlatGeobuf 及常用栅格格式。
- 新增便携式工程打包，可汇总工程及关联数据，便于迁移和交付。
- 新增地图集导出，可按覆盖图层批量输出 PDF，或生成 PNG/SVG 文件包。

#### 地图浏览与选择

- 新增 QGIS 原生地图渲染结果接入和栅格位图缓存，提升复杂图层的显示一致性与平移体验。
- 新增中键平移、以鼠标位置为中心的滚轮缩放、右键地图菜单和地图上下文复制/粘贴。
- 新增单选、多选、矩形选择、套索选择和多边形选择，并支持跨图层空间选择。
- 支持缩放至主选要素或全部已选要素，属性表与地图选择状态保持同步。
- 新增第二地图窗口和视图联动，可并排对照不同图层并同步地图范围。
- 新增数据视图与布局视图切换，布局视图可实时呈现地图框、标题、比例信息和指北针。
- 优化在线底图加载、地图范围变化与渲染刷新逻辑。

#### 编辑、属性与表达式

- 新增持久化编辑会话，支持开始编辑、提交、回滚、撤销和重做，并在退出前提示未保存修改。
- 扩充点、线、面要素新增、节点移动、节点删除、要素删除和切割草图等编辑工具。
- 新增批量属性赋值和多要素选择联动。
- 新增表达式工作台，支持按表达式选择、提取和计算字段，并提供字段、模板与结果预览。
- 新增工程自动恢复快照，异常退出后可恢复工程、图层和待提交编辑状态。

#### 符号、标注与栅格显示

- 扩展矢量渲染模式，新增表达式规则、分类/分级表现及更完整的色带、线型、填充和透明度配置。
- 新增 QGIS QML 样式导入与导出，便于与桌面 QGIS 交换图层样式。
- 完善标注规则、字段表达式、避让、比例尺范围、描边和颜色设置。
- 重构栅格显示面板，补充单波段灰度、RGB 合成、拉伸、色带、NoData、透明色、阴影地形和不透明度设置。
- 改进地图布局导出，可将当前底图和可见图层纳入 PDF、PNG、SVG 输出。

#### 工程、云与系统集成

- 新增 `.qgs` / `.qgz` 工程读取和写出，保存当前图层、工程名称与工程坐标系。
- 新增华为账号登录状态管理，并支持登录信息持久化与退出登录。
- 新增云端工程上传和拉取，可同步工程文本、图层数据包、清单信息及操作日志。
- 新增桌面服务卡片，展示工程名称、图层数、云同步状态、处理状态和地图缩略图。
- 新增系统分享接收、批量 URI 导入、文件关联打开和协同服务入口。
- 新增窗口大小与位置自动保存、全屏/分屏/悬浮窗口模式、自定义标题栏、窗口置顶和关闭保护。
- 应用定位调整为 2in1 设备，并启用云文件同步能力。

#### 工作台与交互体验

- 重构主工作台，新增全局命令搜索、图层/字段搜索、工具搜索和拼音首字母匹配。
- 完善目录面板，加入门户、计算机、收藏、当前工程、最近工程、图层与字段等分区。
- 新增处理任务、云同步、工程交付、拓扑复核、表达式和 XY 表转点等工作台入口。
- 优化内容面板、图层顺序、可见性控制、仅显示当前图层及空状态提示。
- 扩充中文与深色主题资源，统一部分 Material 样式和组件视觉表现。

### 底层与构建改进

- 扩展 ArkTS、N-API 与 C++ GIS Core 三层接口，补充异步任务、工程读写、压缩包解压、空间选择、拓扑、QML、地图渲染和编辑事务能力。
- 大幅扩充 QGIS Core / GDAL 后端算法实现，并将高级空间分析拆分为独立源码模块。
- 更新 CMake 与原生依赖构建脚本，新增 FreeXL 构建流程并调整 GDAL 配置。
- 新增发布包及符号归档产物，便于安装验证和问题定位。

### 兼容性说明

- 应用版本号由 `1.0.0` 更新为 `1.0.3`，`versionCode` 由 `1000000` 更新为 `1000003`。
- 当前设备类型配置由 Phone + 2in1 调整为仅 2in1；如需继续发布手机版，应在发布前恢复并完成手机布局验证。
- 云同步、华为账号、协同服务和部分系统窗口能力依赖设备服务、账号状态及对应系统版本，实际可用性以目标设备为准。
- 当前工作区包含尚未提交的源码、设计稿、发布包和符号文件；正式推送前建议确认大体积发布产物是否通过 GitHub Release 分发，而不是直接纳入 Git 仓库。

## v1.0.0（2026-07-05）

- GeoNest 首个 GitHub 公开版本。
- 提供基础 GIS 工程工作台、矢量/栅格图层加载、地图浏览、属性查看、基础符号与标注配置。
- 集成 QGIS Core、GDAL/OGR、PROJ、GEOS、Qt6 与 HarmonyOS N-API 桥接。
- 提供首批 22 项 Processing 算法和基础图层处理能力。
- 完成开源仓库说明、许可证及第三方依赖合规文档。
